# 🎂 Peringatan Tarikh Lahir — SK Kiandongo

Aplikasi web untuk mengurus, menjejak, dan menghantar peringatan hari lahir bagi guru, staf, dan murid.

- Senarai kenalan

- Backup & eksport data
